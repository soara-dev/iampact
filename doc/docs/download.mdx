# Download

Sebagai pengguna, Anda dapat memanfaatkan file yang telah disediakan untuk integrasi dengan proyek Anda. Di bawah ini adalah opsi unduhan untuk Iampact:

### Compiled Iampact

Unduh Iampact terbaru yang sudah siap pakai untuk integrasi langsung ke proyek Anda.

{/* prettier-ignore */}
<button type="button"
  className="button button--primary button--lg custom-button"
  target="_blank"
>Download</button>

### CDN Iampact

Jika Anda lebih suka menggunakan CDN, Anda dapat langsung menyertakan file Iampact dari server CDN. Cukup tambahkan kode berikut ke dalam file HTML Anda:

```html
<script src="https://iampact.smactactic.dev/dist/main.js"></script>
```
