# Fill Form

Pada saat melakukan edit data terkadang kita harus melakukan set data ke dalam input secara satu persatu agar data dapat di edit oleh pengguna. Developer dapat melakukan set value kedalam inputan secara otomatis menggunakan fillForm. Berikut adalah contoh penggunaan fillForm pada aplikasi

### Menampilkan Data ke Inputan

Pada iampact anda dapat langsung menampilkan data ke inputan form secara langsung dengan fillForm, fillForm akan otomatis mencari input name yang sama dengan key pada object yang akan di set.

### Menambahkan Pengecualian

Pada saat melakukan edit data terkadang kita tidak ingin ada data yang di tampilkan di dalam input contohnya seperti password. pengguna harus mengisi ulang password. kita dapat menggunakan param except pada fillForm.

```js
const data = {
  name: "Harithya Wisesa",
  email: "admin@gmail.com",
  phone: "081234567890",
  password: "di enkripsi nih bosss",
};

http.fillForm(data, {
  except: ["password"],
});
```

### Memberikan Callback 
Pada beberapa kasus, kita mungkin ingin melakukan aksi tambahan setelah data berhasil diisi ke dalam form. Kita dapat menggunakan parameter callback pada fillForm untuk menjalankan fungsi tertentu setelah proses pengisian form selesai.

```js
http.fillForm(data, {
  callback: () => {
    console.log("Form telah diisi dengan data!");
  },
});
```
